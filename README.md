# Sports-Fandom-Data-Analysis-and-Visualization
Developed an interactive data visualization project using Tableau and Python to analyze favorite sports teams across Indian states. Processed data from surveys, social media, merchandise sales, and attendance records to identify trends influenced by geography, demographics, and local culture.
